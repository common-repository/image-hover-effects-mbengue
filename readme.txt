== Description ==
Image Hover Effects Mbengue is a plugin to generate a hover effect on an image. On mouse hover the image is replaced by another.
Image Hover Effects Mbengue is responsive and adapts to any editor including Gutenberg.
Demo : https://www.zake.pro/test-hover/

== Installation ==
1 - Install Plugin
2 - Activate Plugin
3 - Create yours \"Hovers Rollovers\" in menu left top
4 - Copy and paste your shortcode in your editor.Eg.: [HOVERMBENGUE id=\"21265\"]
